# ⚡ Ismath AI – SaaS Landing Page

This is my 3rd frontend project — a modern AI SaaS landing page built using HTML and CSS.

It is a single-page website designed with a clean and modern UI to showcase an AI-based SaaS concept.

## 🧩 Project Overview
This project represents a simple SaaS landing page that focuses on design, structure, and layout.  
It includes multiple sections that represent a real-world SaaS product landing page.

## 🚀 Features
- Modern Hero Section with AI branding  
- Features section with structured cards  
- AI Solutions section with clean layout  
- Contact form UI design  
- Project preview section  
- Responsive and modern styling  

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Google Fonts (Inter)

## 📚 Learning Outcomes
Through this project, I learned:
- How to structure a professional landing page  
- How to use CSS Grid for layouts  
- How to manage spacing and alignment  
- How to design modern SaaS-style UI  
- How to build reusable components
